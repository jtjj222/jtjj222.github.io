Cheap VR Motion Tracking

Using lasers to scan across the room, the position of a Google Cardboard VR headset is calculated by timing how long it takes for the lasers to hit an Arduino. I hope to acheive most of the effect of an HTC Vive, with one tenth the cost. In order to solve the problem of keeping the scan rate constant, I plan to use an IR emiter and receiver to synchronize the laser to the Arduino. Source

Open Source Remote Support

While at the YMCA, I developed a open-source remote support tool for John Island Camp using Java and UDP hole-punching. I needed quick and simple ssh access. It used Netty and utp to tunnel multiple tcp connections over the network, and a small discovery server to handle connecting clients. It used less bandwidth and was easier to set up than off the shelf solutions. Source
