Open Source Contribution: I am working with the maintainer of the popular AnySoftKeyboard project to implement Gesture Typing, similar to the stock Google keyboard. This is the first Open Source gesture keyboard available for Android. My contributions include the code that generates simulated paths and matches them to the user input.Source

Cheap VR Motion Tracking

Work in progress: I am working to create a cheap, DIY alternative to the HTC Vive using Google Cardboard and Arduino. Using mirrors to scan line-lasers across the room, the position of the controller can be calculated by timing how long it takes for the laser to reach a photoresistor. To synchronize the start of the laser sweep on the emitter and receiver, I plan to use an off-the-shelf IR LED and receiver.Source

SACY Lending Library

Created a web application to track clients, donations and client visits for the Sudbury Action Centre for Youth. Using Python and Django, it produced reports to gain insight into how clients use their services. Accurate data allowed SACY to seek funding, find areas to improve, and eliminate manual labour, allowing them to focus on the already-difficult task of helping at-risk youth. I collected requirements from non-technical users, and iterated frequently to ensure the application created value.

Remote Support

While at the YMCA, I developed a open-source remote support tool for John Island Camp using Java and UDP hole-punching. I needed quick and simple ssh access through a NAT, without port forwarding. It used Netty and utp to tunnel multiple tcp connections over the network, and a small discovery server to handle connecting clients. It used less bandwidth and was easier to set up than off the shelf solutions. Source
